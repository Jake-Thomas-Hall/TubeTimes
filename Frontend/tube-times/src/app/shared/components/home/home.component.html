Welcome to Tube Times

A tool to find information about tube services

View live status

You can use this tool to look at the current status for all of the rail based services in the TfL network.

Note that each line may contain a list of statuses, which affect different parts of the line. You can filter the statuses by type at the top, and search by name in the provided search box.

View

View routes

With each line, they often consist of different routes, on these pages you're able to select the route and direction you wish to view for a line.

You can click on a station to view live updates of next services that are stopping at that station.

If the line you want isn't below, simply click "View all" and you can then select to view routes for a line.
